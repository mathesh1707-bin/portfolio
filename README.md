# 💼 Mathesh S – Portfolio

A full-stack personal portfolio website showcasing my projects, skills, and development journey. Built with a focus on clean UI, performance, and real-world backend integration.

## 🚀 Features
- Modern and responsive UI
- Interactive sections with smooth animations
- Projects and skills showcase
- Contact form with backend email integration
- Auto-reply system using Nodemailer

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Email Service: Nodemailer (Gmail SMTP)

## 📬 Contact System
Users can send messages through the portfolio, which are delivered directly to my email. An automated confirmation email is also sent back to the user.

## 📁 Project Structure
- `/frontend` – UI and client-side logic
- `/backend` – Express server and email functionality
